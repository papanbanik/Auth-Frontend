This is complete **authentication system**. Writing clean, maintainable, and scalable code by clean code principles.

## Features
- Token-based authentication (JWT)
- User registration (sign up)
- Secure login
- Email verification with OTP
- Forgot password and password reset
- Update user profile
- Google OAuth login (separate authentication flow)
- 
**Tech Stack**
 - Next.js, TypeScript, Node.js, Express.js, MongoDB, JWT Authentication, Google OAuth, Nodemailer (for email verification)
